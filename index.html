Most-Popular-Stories---Compatibility-version

Android Application. Extracts the most shared and most read stories from popular news websites. So far most of the implementation is in place for the BBC News section. Still in progress.

IMPORTANT: requires having android-support-v7-appcompat library project in the same parent folder/workspace as this project. More information at http://developer.android.com/tools/support-library/setup.html. Otherwise use the higher-API version (to be added to my account shortly)

Known Bug: If there is no internet connection on device, the app will crash. This will be fixed. Also, the consistency between activity UIs needs to be improved, preferably by having more functionality in the base activity than presently.
